The bin folder contains the .exe
You must download and install http://www.microsoft.com/en-gb/download/details.aspx?id=40784 to run the exe

The source code is in SFML_Project.

The source code shows task 1 and 3, task 1 is displayed on the console and task 2 is a graphical display
that is visible to play the game graphically with 2 players. The score will be shown in the console.
The source code is written for c++11 compilers specifically and will not compile on earlier versions
